Racing Simulator Guide

Welcome! This is a multi-series racing simulator. Here's how it works:

Getting Started

  1. Select a series from the top bar (Cup, Xfinity, Truck, F1, IndyCar, Custom F1, Custom NASCAR).
  2. Pick a track from the dropdown below.
  3. Click Qualify to set the starting grid, then Start Race to run it.
  4. View Standings and Results via the tab bar.

Race Controls

  • Speed buttons (1x / 3x / 10x / 25x / Full) control simulation speed.
  • Auto mode automatically runs Qualify + Race for you.
  • Pause pauses the race; click Resume to continue.
  • Stages toggle enables stage cautions and stage points (NASCAR series only).
  • Chase toggle enables the playoff/chase format (NASCAR series only).

Schedule Editor

Click Schedule to open the schedule editor. You can add/remove tracks, reorder races (drag via arrows), save presets, and enable Free Pick mode to ignore the schedule and pick any track each race.

Standings

  • Click column headers to sort (Points, Wins, Avg Finish, etc.).
  • Team Standings toggle shows combined points per team.
  • During the Chase, Season View toggle switches between chase and regular season points.
  • Use Reset Season to clear standings and results (keeps schedule).
  • Use New Season to reset everything including the schedule.

Editors

  • Rosters — Edit driver names, numbers, teams, manufacturers, and ratings (OVR, SS, IT, ST, RC, AGG, CON, CAR). Driver changes take effect immediately without resetting the season.
  • Tracks — Add, edit, or delete tracks from the database. Track ID is used in schedules.
  • Points — Edit the points awarded per finishing position. Also adjust Chase format (regular season length, chase field size).
  • Export CSV — Downloads standings and race results as a CSV file for spreadsheets.

Cloud Save (Firebase)

  • Sign in with Google to sync your data (rosters, seasons, presets, tracks, points) to the cloud.
  • Data syncs automatically on save. Sign in on another device to restore.

Series Differences

  • NASCAR series (Cup, Xfinity, Truck, Custom NASCAR): Stages enabled, Chase/playoff format, 36–40 car fields.
  • Open-wheel series (F1, IndyCar, Custom F1): No stages, no chase, formula-style points, different physics and pit strategies.
